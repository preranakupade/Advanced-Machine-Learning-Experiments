# Advanced-Machine-Learning-Experiments

This repository contains a collection of **Machine Learning** and **Deep Learning** experiments implemented using **Python**.
It is designed for academic practicals, hands-on learning, and showcasing AI/ML skills through real implementations.


## Experiment List

### Experiment 1: Ensemble Learning

Implementation of ensemble methods to improve prediction performance by combining multiple models.

### Experiment 2: Bagging & Boosting

Implementation of Bagging and Boosting techniques such as AdaBoost.

### Experiment 3: Random Forest

Classification and prediction using Random Forest algorithm.

### Experiment 4: Markov Decision Process & Bayesian Learning

Implementation of:

* Markov Decision Process (MDP)
* Bayesian Learning / Naive Bayes Classifier

### Experiment 5: Recommendation System

Implementation of:

* Collaborative Filtering
* Content-Based Recommendation

### Experiment 6: Perceptron Model

Implementation of Perceptron algorithm for binary classification.

### Experiment 7: CNN Image Classification

Train a Convolutional Neural Network (CNN) on image datasets for classification.

### Experiment 8: CNN Video Classification

Train a CNN model on video datasets and evaluate performance.

### Experiment 9: RNN Text Summarization

Generate summary of documents using Recurrent Neural Network (RNN).

### Experiment 10: RNN Sentiment Analysis

Classify sentiment of text data using RNN model.


## 🛠️ Technologies Used

* Python
* NumPy
* Pandas
* Matplotlib

* TensorFlow / Keras
